# 📊 微信朋友圈分析工具

## 📌 项目简介

本项目旨在将“发朋友圈”这一日常社交行为进行量化建模，构建一个**收益-风险评估系统**，帮助用户在发布内容前进行理性决策。

通过对受众结构、关系复杂度以及内容特征的分析，系统可以估算一条朋友圈可能带来的**正向收益与潜在风险**，并给出是否建议发布的判断。

---

## 🧠 核心思想

将朋友圈发布行为抽象为一个效用函数（Utility Function）：

U = Σ p_i (B_i - C_i · k_i)

其中：

- p_i：第 i 类受众占比  
- B_i：该类受众带来的收益（认同、互动、形象提升等）  
- C_i：潜在成本（误解、嫉妒、隐私暴露等）  
- k_i：关系复杂度（关系越复杂，风险放大系数越高）

最终 U 表示发布该朋友圈的**净收益评分**。

---

## 🎯 项目目标

- 📌 将主观社交决策转化为可计算模型

- 📌 提供“发 / 不发 / 分组可见”的决策建议  
- 📌 探索社交行为的量化分析方法

## ⚙️ 功能设计（MVP）

### 1️⃣ 朋友圈内容输入

用户可输入或选择：

- 内容类型（如：日常记录 / 成就展示 / 情绪表达 / 旅游 / 健身等）
- 是否包含敏感信息（如：金钱、情绪、关系、定位等）

---

### 2️⃣ 受众结构建模

用户自定义受众分类及占比，例如：

- 家人
- 亲密朋友
- 普通朋友 / 同学
- 不熟联系人
- 特殊对象（如前任、暧昧对象等）

---

### 3️⃣ 收益与风险评估

对每类受众设定：

- 收益 B_i：
  - 获得认同  
  - 增强关系  
  - 展示形象  

- 成本 C_i：
  - 被误解  
  - 引发比较 / 嫉妒  
  - 信息泄露  

- 关系复杂度 k_i：
  - 简单关系（低）  
  - 中等关系  
  - 复杂关系（高）

---

### 4️⃣ 评分与结果输出

系统输出：

- 📊 总收益评分（Utility Score）  
- ⚠️ 风险贡献分析（按人群拆分）  
- 💡 发布建议：
  - ✅ 建议发布
  - ⚠️ 谨慎发布
  - ❌ 不建议发布

---

### 5️⃣ 可视化（可选扩展）

- 雷达图（收益 vs 风险）
- 柱状图（不同人群影响）
- 饼图（受众结构）

## ⚠️ 项目定位说明

本项目为**决策辅助工具**，并非真实预测系统。

其输出结果基于用户输入参数和建模假设，仅用于：

- 提供思考框架  
- 辅助判断风险  
- 提升表达策略  

不保证对现实社交结果的准确预测。

---

## 📖 项目意义

在社交媒体环境中，信息传播具有不确定性与复杂性。本项目尝试通过建模方式：

- 理解“表达”与“风险”的关系  
- 探索人际互动中的隐性成本  
- 提供一种理性化的社交决策思路
